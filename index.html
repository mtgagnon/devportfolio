Bol: Fall 2023

A unique take on a puzzle platformer where the player moves the platforms using different kinds of balls. Created by myself in Unity using C#, I designed 6 levels made to challenge the player, while teaching them new mechanics. Challenged game designed skills to create something that is intuitive for players but also has a smaller scope so I could get the game done in time. The project took 2 weeks from ideation to realization.

Zelda Remake: Fall 2023

A recreation of the original NES Zelda game, with a twist. Created in Unity using C# with a teammate. I managed the implementation of player and enemy movement, room transitions and room function such as locking on entering. I also created projectiles that could be fired by Aquamentus, these were later used again for a new freeze ray weapon which I also implemented.

BrainHQ Posit Science: Summer 2023

Initiated the design, implementation, and migration to a new low latency, multi-threaded, sound engine using the Oboe sound library in C++. Used knowledge of computer architecture and thread management to achieve a balance between high-quality audio and low latency. Independently learned how to use the Oboe Sound Library to integrate it with the current Android app and create readable, production level code that can be maintained by other people.

Network File Server: Fall 2022

Implemented a multi-threaded network file server for client interaction via network messages using C++. Deepened grasp of hierarchical file systems, socket programming, client-server architectures, and network protocols. Constituted a significant multi-threaded application with fine-grained locking.
